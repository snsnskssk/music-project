## 작품개요
이 작품은 라즈베리파이를 활용하여 거리에 따라 다른 음계를 생성하는 악기를 만드는 것을 목표로 한다.
## 작품제작에 사용한 센서와 엑추에이터
-초음파 센서 : 거리를 측정해 준다.
-버튼 : 누르면 동작을 종료한다.
-부저 : 음을 낸다.
